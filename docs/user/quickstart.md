# Guide de Démarrage Rapide - ArchiMap

## Bienvenue sur ArchiMap

ArchiMap vous permet d'explorer et comparer les communes françaises selon différents critères pour trouver l'endroit idéal où vivre.

## Navigation de Base

### Explorer la Carte

1. **Zoom**: Utilisez la molette de la souris ou les boutons +/- en haut à droite
2. **Déplacement**: Cliquez et glissez pour vous déplacer sur la carte
3. **Niveaux**:
   - Vue éloignée: Régions
   - Vue moyenne: Départements
   - Vue rapprochée: Communes

### Sélectionner un Critère

1. Dans la barre latérale gauche, cliquez sur une **catégorie** (Climat, Coût de Vie, etc.)
2. La catégorie s'ouvre pour révéler ses critères
3. Cliquez sur un **critère** pour le voir sur la carte
4. La carte se colore selon les valeurs du critère

### Lire la Carte

- Les **couleurs** indiquent les valeurs du critère
- La **légende** en bas de la barre latérale explique l'échelle
- **Survolez** une zone pour voir son nom et sa valeur

## Activer Plusieurs Critères

1. Cochez la case à côté de chaque critère que vous voulez voir
2. Les couches se superposent avec transparence
3. L'indicateur "Couches actives" affiche le nombre de critères activés
4. Décochez pour désactiver une couche

## Naviguer dans la Hiérarchie

1. **Cliquez** sur une région pour zoomer dedans
2. **Cliquez** sur un département pour voir ses communes
3. Utilisez le **fil d'Ariane** en haut pour revenir en arrière
4. Le bouton **"France"** vous ramène à la vue nationale

## Rechercher une Commune

1. Cliquez sur la barre de recherche en haut
2. Tapez le nom de la commune (minimum 3 lettres)
3. Sélectionnez dans les suggestions
4. La carte centre automatiquement sur la commune

## Mode Sombre

- Cliquez sur l'icône lune/soleil en haut à droite
- La carte et l'interface s'adaptent automatiquement

## Partager une Vue

1. L'URL change automatiquement quand vous naviguez
2. Copiez l'URL pour partager votre vue actuelle
3. Le destinataire verra exactement la même carte et les mêmes critères

## Raccourcis Clavier

| Touche | Action |
|--------|--------|
| `+` / `-` | Zoom avant / arrière |
| `Échap` | Fermer les panneaux |
| `/` | Focus sur la recherche |
| `D` | Basculer mode sombre |

## Besoin d'Aide?

- **Documentation complète**: `/docs`
- **Signaler un problème**: Bouton "Signaler une erreur" dans les détails d'une commune
